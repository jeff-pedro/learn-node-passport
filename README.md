# Exploring Passport.js

This repository is used to follow the tutorial available through Passport, where a "**Todo**" app with sign in functionality is built using [Express](https://expressjs.com/), [Passport](https://www.passportjs.org/) and [SQLite](https://www.sqlite.org/).

## Steps
* [Login Prompt](https://github.com/jeff-pedro/learn-node-passport/tree/login-prompt)
* [Verify Password](https://github.com/jeff-pedro/learn-node-passport/tree/verify-password)
* [Establish Session](https://github.com/jeff-pedro/learn-node-passport/tree/establish-session) 
* [Log Out](https://github.com/jeff-pedro/learn-node-passport/tree/logout) 
* [Sign Up](https://github.com/jeff-pedro/learn-node-passport/tree/signup)
